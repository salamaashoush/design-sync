# Tokenize

Figma plugin and a set of utils to tokenize your design system

## Development

### Install dependencies

```bash
pnpm install
```

### Run the dev server

```bash
pnpm run dev
```

### Build the plugin

```bash
pnpm run build
```

### Load the plugin into Figma

Load the `manifest.json` file in the `apps/plugin/dist` folder into Figma.
